Установка
npm install -g newman-reporter-allure
использование
Чтобы сгенерировать результаты Allure, укажите allureв Newman -r или --reporters опцию.

newman run <Коллекция> -e <Среда> -r allure
newman run <Коллекция> -e <Среда> -r allure --reporter-allure-export <Путь к файлам отчета>
Создание и обслуживание отчета Allure
Результаты Allure будут созданы в папке allure-results в корневом каталоге. Используйте командную строку allure для локального обслуживания отчета.

allure serve
Создайте папку веб-приложения статического отчета с помощью командной строки allure

allure generate --clean
